1. 주제 : 무신사(옷쇼핑)
1-1. 시나리오
   1. 고객이 옷을 골라 주문한다.
   2. 주문이 완료되면 배송을 시작한다.
   3. 배송이 시작되면 상태가 변경된다.
   4. 배송이 완료되면 전체수량이 주문수량만큼 줄어든다.
   5. 고객의 변심으로 옷을 취소한다.
   6. 주문이 취소되면 배송이 취소된다.
   7. 배송이 취소되면 상태가 변경된다.
   8. 배송이 취소되면 전체수량이 취소수량만큼 증가한다.
